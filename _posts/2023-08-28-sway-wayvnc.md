十分[怀念](2016/07/19/i3-wm.html)i3wm。macOS上切来切去效率很低。

以下办法可以无头(headless)地联上Linux主机，效率还不错。随身用树莓派办公好了。

sway是i3在wayland上的实现，可能会比较快吧。wayland是X11的替代。
wayvnc是wayland的vnc。


```bash
$ sudo aptitude install xwayland sway wayvnc

$ XDG_RUNTIME_DIR=/tmp/ WLR_BACKENDS=headless WLR_LIBINPUT_NO_DEVICES=1 sway &
$ swaymsg --socket /tmp/sway-ipc.*.sock exec 'WAYLAND_DISPLAY=wayland-1 wayvnc'
$ swaymsg --socket /tmp/sway-ipc.*.sock output "HEADLESS-1" resolution 1920x1080

$ ssh -L 9901:127.0.0.1:5900 your_host
```

然后VNC Client联`127.0.0.1:9901`即可

 - macOS Client: Real VNC Viewer
 - **FIXME:** macOS原生的`vnc://`不知道为啥不好使

好处:
 - 不引入额外的认证方式，全靠ssh


## i3 (sway) 配置

1. Windows键似乎在Real VNC客户端里不好使，要配成Alt (Mod1)。
2. sway的default terminal是foot

## Troubleshoot

如果 RealVNC VNC Viewer 报错:

> ZlibInstream: inflate failed with error "-3"

参考: <https://www.moonfly.net/archives/52.html>

> 只需要到该会话的连接属性中修改一下要显示的颜色质量就可以了; 
> 默认是“自动”模式就会出现上述的问题，只需要手动修改成“高，中，低”即可；
